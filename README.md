# SafeCampus Complaint System
AI-powered campus safety and ragging complaint management system with blockchain security and AI-based severity analysis.

## Features

* Secure student complaint reporting
* AI severity analysis for complaints
* Blockchain-based complaint security
* Authentication and authorization
* Admin complaint management
* Real-time monitoring dashboard
* Complaint tracking system

## Technologies Used

* React.js
* Node.js
* Express.js
* MongoDB
* Python
* Blockchain Integration
* JWT Authentication

## Project Structure

* Frontend - React application
* Backend - Node.js/Express API
* Python API - AI severity analysis
* Blockchain Module - Complaint integrity and security
